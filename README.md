# Advanced To-Do Manager 📝

A Python-based **To-Do List application** with **GUI** and advanced features for managing your tasks efficiently.  

---

## Features

- ✅ **Login & Register System** – Securely manage users.  
- 🎨 **Dark Mode GUI** – Easy on the eyes with a sleek dark theme.  
- 📌 **Task Management** – Add, delete, and complete tasks.  
- 🔎 **Search & Filter** – Quickly find tasks by keywords.  
- 📊 **Task Statistics Dashboard** – View total, completed, and pending tasks.  
- ⭐ **Task Priorities** – Mark tasks as Low, Medium, or High priority.
